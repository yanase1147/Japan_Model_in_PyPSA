<h1>バイオマス発電</h1>
<h2>1. なぜ通常はディスパッチされないのか？</h2>
バイオマス発電が「垂れ流し（一定出力）」で運用される主な理由は3つあります。
<h3>経済的理由（FIT制度）:</h3>
日本の多くのバイオマス発電所は固定価格買取制度（FIT）を利用しています。発電すればするほど収益が増えるため、意図的に出力を下げる（ディスパッチする）動機がありません。<br>
<h3>技術的制約:</h3>
特に「木質バイオマス」などの燃焼設備（ボイラー）は、火力を急激に上げ下げするのが苦手です。出力を頻繁に変えると、燃焼効率が落ちたり、設備を傷めたりするリスクがあります。<br>
<h3>燃料供給の安定性:</h3>
バイオマスは燃料（木くず、パーム殻など）を燃やし続ける必要があるため、ガスタービンのようにスイッチ一つで即座に調整するような運用には向きません。<br>

<h2>2. ディスパッチされる「例外」ケース</h2>
ただし、以下の状況では出力が調整されることがあります。
電力需給逼迫時・再エネ抑制時:
太陽光や風力の発電量が増えすぎて電力が余った場合、優先順位に従って「出力制御」が行われます。バイオマスは火力発電の次に制御対象となるため、地域によっては出力を抑えるよう指示（ディスパッチ）されることがあります。
FIP制度への移行:
2022年から始まったFIP制度（市場価格に連動してプレミアムがつく制度）を選択している発電所の場合、価格が高い時間帯に多く発電し、安い時間帯に抑えるといった「市場主導のディスパッチ」を行うインセンティブが生まれます。
バイオガス発電:
木質バイオマスと違い、家畜排泄物などを発酵させる「バイオガス発電」は、ガスタンクに貯蔵ができるため、電力需要に合わせてエンジンを回す（ピークシフト）といった調整が比較的容易です。
